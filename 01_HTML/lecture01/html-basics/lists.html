Lists in HTML

There is 2 type of list in HTML5 :-

  • ordered list
  • unordered list

Unordered List

For creating the unordered lists we use ul tag and between the ul tag, li tag is used for making unordered list elements. In the ul and li tag there is an attribute named type which have 3 values :

  • disc
  • circle
  • square

Ordered List

For creating the ordered lists we use ol tag and between the ol tag, li tag is used for making ordered list elements. In the ol and li tag there are also two attributes named type and start which can have following values :

  1. type="1"
  2. type="A"
  3. type="i"
  4. type="a"

    5. Note : In the start attribute we will write that from where I want to start that particular numbering. Ex - start="4"
